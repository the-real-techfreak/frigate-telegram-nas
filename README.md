# frigate-telegram-nas

This repo is created for installing and setting up Frigate NVR, enable an automation to save the frigate "review" clips (Review -> compilation of simultaneous events) in a particular folder and then send this clip to the user through Telegram Bot - Send Video service

The setup is tested on a Debian distro installed on a Proxmox container.

Hardware for this setup include a tiny PC (HP Elitedesk 800 G3 mini with Intel Core i5) and a Coral TPU
